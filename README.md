# vlsp2023

## Set up

**Create a new virtual enviroment (Python>=3.10) and activate it**

```bash
    python -m venv venv
    source venv/bin/activate
```
**Install requirements**

```bash
    pip install -r requirements.txt
```

## Demo

**Backend**
```bash
    cd backend
    python main.py
```
**Frontend**
```bash
    cd frontend
    python [task1/task2].py
``` 

## Docker
**Build**
```bash
    docker compose build
```
**Run**

```bash
    docker compose up -d
```
